Float & clear

Floated elements are removed from the normal flow but influence the surrounding context.

"h3 floating"

Floated elements stay within the content area of the element that contains it.

Elements don't float higher than their reference in the source.

"h3 floating"

"h3 not clearer"

"h3 clearer left"

You must provide a width for a floated text element.

Clear prevents an element from appearing next to a floated element and forces it to start against the next available "clear" space
